# IPL Data Science Dashboard

An interactive D3.js dashboard for exploring Indian Premier League cricket analytics, featuring team performance, player statistics, and match predictions.

## 🚀 Live Demo
[View Dashboard](https://yourusername.github.io/ipl-data-science-project/)

## 📊 Features
- **Overview**: Key metrics and toss decision analysis
- **Team Explorer**: Win/loss records by team
- **Analytics**: Top teams, run scorers, and wicket takers
- **Player Stats**: Individual player performance
- **Match Predictor**: Simulate match outcomes

## 🛠️ Tech Stack
- **Frontend**: D3.js, Bootstrap 5
- **Data Processing**: Python, Pandas
- **Deployment**: GitHub Pages

## 🔧 Local Development
1. Clone the repository
2. Open `index.html` in a web browser
3. All features work offline

## 📈 Data Pipeline
The `src/` folder contains the complete data processing pipeline:
- Data collection and cleaning
- Feature engineering
- Model training (for future enhancements)
